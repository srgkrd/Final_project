# Проект Проект: разработка автотеста
Проверяется, что по треку заказа можно получить данные о заказе в мобильном  приложении "Яндекс Самокат вход для курьера"

# Используемые инструменты:
- Среда разработки PYcharm
- Phyton
- Postman
# Установка и запуск:
- Установите необходимые пакеты, перейдя в вкладку "Python Packages"
- Python Packages:"requests","pytest"
- Создайте курьера с помошью Postman
- Запустите тесты тесты в pytest, фаил "test_create_order_and_chek_order_by_track.py"

# Проект включает в себя следующие файлы:
- test_create_order_and_chek_order_by_track.py
- sender_stand_request.py
- Data.py
- configuration.py
